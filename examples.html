Examples

Code style check

If you have a bigger code base it makes sense to only check staged files for code style violations. 

{
  "pre-commit": {
    "enabled": true,
    "actions": [
      {
        "action": "phpcs --standard=psr12 {$STAGED_FILES|of-type:php}",
        "conditions": [
          {
            "exec:"\\CaptainHook\\App\\Hook\\Condition\\FileStaged\\OfType",
            "args": [
              "php"
            ]
          }
        ]
      }
    ]
  }
}

Automatic composer install

Run composer install automatically so you don't forget to update your dependencies after pulling changes or checking out a different branch. 

{
  "post-merge": {
    "enabled": true,
    "actions": [
      {
        "action": "composer install",
        "conditions": [
          {
            "exec:"\\CaptainHook\\App\\Hook\\Condition\\FileChanged\\Any",
            "args": [
              ["composer.json", "composer.lock"]
            ]
          }
        ]
      }
    ]
  },
  "post-checkout": {
    "enabled": true,
    "actions": [
      {
        "action": "composer install",
        "conditions": [
          {
            "exec:"\\CaptainHook\\App\\Hook\\Condition\\FileChanged\\Any",
            "args": [
              ["composer.json", "composer.lock"]
            ]
          }
        ]
      }
    ]
  }
}
